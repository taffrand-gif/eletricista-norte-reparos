    📋 Como Escolher

    • Temperatura de cor: 2700K (quente) para sala/quarto, 4000K (neutro) para cozinha/escritório
    • IRC (Índice Reprodução Cromática) >80 obrigatório
    • Equivalência real: verifique lúmens, não watts
    • Regulável (dimmable) se usar dimmer
    • Vida útil: mínimo 10000h (boas marcas dão 15000-25000h

    🏆 A Nossa Recomendação

    Philips LED A60 para iluminação geral — qualidade consistente a preço acessível. Para spots, Philips GU10. Compre em pack de 6 para poupar.
