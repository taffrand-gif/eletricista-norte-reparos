Melhores Disjuntores para Casa 2026

Disjuntores e diferenciais protegem a instalação e as pessoas. Escolher os corretos é fundamental para a segurança da casa.

#1 — Schneider Electric iC60N

Referência em Portugal. Curva C para uso residencial. 6kA de poder de corte. Fiável e fácil de encontrar.

Preço aproximado: ~8-12€/unidade

🛒 Ver na Amazon.es →

#2 — ABB S200 Série

Qualidade suíça. Compacto. Indicador de estado claro. Curva B ou C disponível. Muito usado em Europa.

Preço aproximado: ~10-15€/unidade

🛒 Ver na Amazon.es →

#3 — Schneider Diferencial iID 40A 30mA

Diferencial tipo AC para proteção de pessoas. 30mA é o padrão residencial. Botão de teste integrado.

Preço aproximado: ~35€

🛒 Ver na Amazon.es →

📋 Como Escolher

  • Curva C para uso geral residencial
  • Curva B para circuitos sensíveis (informática)
  • Diferencial 30mA obrigatório por lei
  • Poder de corte mínimo 6kA
  • Marca reconhecida: Schneider, ABB, Legrand, Hager

🏆 A Nossa Recomendação

Schneider iC60N para disjuntores + Schneider iID para diferenciais. É o que usamos nas nossas instalações. Fiável e disponível em todo o lado.
