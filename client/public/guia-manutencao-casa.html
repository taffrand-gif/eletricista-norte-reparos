Em Trás-os-Montes, as casas têm particularidades: - Invernos rigorosos (até -10°C) - Sistemas de água antigos - Instalação elétrica frequentemente antiga - Pouca oferta de profissionais 24h Uma manutenção preventiva custa **200-500€/ano**. Uma reparação de emergência custa **500-2.000€** + danos colaterais. **Este guia ajuda-o a evitar as surpresas.** --- ## 🌸 PRIMAVERA (Março-Maio) ### Canalização - [ ] Verificar todas as torneiras exteriores (preparação para uso intensivo) - [ ] Testar autoclismos - substitui o mecanismo se tiver mais de 5 anos - [ ] Inspecionar o estado do telhado e caleiras (limpar folhas) - [ ] Verificar a pressão da água (se >5 bar, instalar redutor) - [ ] Limpar/arejar arejadores das torneiras (calcário acumulado durante o inverno) ### Eletricidade - [ ] Testar todos os disjuntores diferenciais (botão "T" - deve disparar) - [ ] Inspecionar cabos elétricos exteriores (após o Inverno) - [ ] Limpar filtros de AC / desumidificador - [ ] Verificar iluminação do jardim --- ## ☀️ VERÃO (Junho-Agosto) ### Canalização - [ ] Aumentar a frequência de verificação de fugas (rega frequente) - [ ] Manter atenção a cheiros estranhos (fossa sética cheia com calor) - [ ] Verificar estado dos esquentadores (uso intenso) - [ ] Limpar filtros da máquina de lavar loiça - [ ] Purgar o termoacumulador (1x/ano) ### Eletricidade - [ ] Verificar ventilação dos armários de contadores - [ ] Inspecionar AC antes da época de calor - [ ] Verificar disjuntores após trovoadas - [ ] Limpar filtros de AC --- ## 🍂 OUTONO (Setembro-Novembro) ### Canalização - [ ] **ISOLAR CANOS EXTERIORES** (esta é a prioridade #1 do outono!) - [ ] Purgar sistema de rega - [ ] Fechar válvulas de água exteriores - [ ] Inspecionar telhado antes das chuvas - [ ] Verificar caleiras e algerozes - [ ] Limpar fossa sética antes do Inverno ### Eletricidade - [ ] Verificar iluminação exterior (mais horas de escuro = mais uso) - [ ] Testar aquecimento elétrico antes do frio - [ ] Inspecionar instalação elétrica após trovoadas - [ ] Verificar estado dos cabos do jardim --- ## ❄️ INVERNO (Dezembro-Fevereiro) ### Canalização - [ ] **MANTER TEMPERATURA > 10°C** em zonas com canalização - [ ] Abrir portas de armários com canos (calor da casa chega) - [ ] Deixar uma torneira a gotejar se risco de congelamento (1 gota/seg) - [ ] Verificar telhado depois de tempestades - [ ] Fechar a válvula geral se a casa ficar vazia ### Eletricidade - [ ] Aumentar a frequência de inspeção (uso intensivo) - [ ] Verificar estado do aquecimento elétrico - [ ] Não sobrecarregar extensões múltiplas - [ ] Testar luzes de emergência --- ## 🚨 QUANDO CHAMAR PROFISSIONAL ### CANALIZAÇÃO - Ligue IMEDIATAMENTE se: - 🔴 Cano rebentado com perda de água - 🔴 Sem água quente no Inverno (especialmente com crianças/idosos) - 🔴 Inundação (risco estrutural) - 🔴 Fossa sética com refluxo - 🔴 Cheiro a gás (evacuar a casa) ### ELETRICIDADE - Ligue IMEDIATAMENTE se: - 🔴 Cheiro a queimado no quadro elétrico - 🔴 Faíscas em tomadas - 🔴 Disjuntor que não para de disparar - 🔴 Sem luz em toda a casa (verificar vizinhos primeiro) - 🔴 Aparelho a dar choque --- ## 💰 CUSTOS MÉDIOS NORTE REPAROS 2026 ### Canalizador | Serviço | Preço a partir de | |---------|---------------------| | Desentupimento | 110€ | | Busca de fuga | 150€ | | Esquentador (reparação) | 150€ | | Reparação urgente | 110€ | | Autoclismo | 80€ | | Torneira | 80€ | ### Eletricista | Serviço | Preço a partir de | |---------|---------------------| | Avaria urgente | 110€ | | Diagnóstico | 90€ | | Quadro elétrico novo | 280€ | | Iluminação LED | 75€ | | Certificação DGEG | 180€ | **IVA isento** (art.º 53.º CIVA) na maioria dos casos. **Transparência total** -
