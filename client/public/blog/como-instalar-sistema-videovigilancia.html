    Como Instalar Sistema de Videovigilância

    Um sistema de videovigilância permite monitorizar a casa 24h por dia, mesmo à distância pelo telemóvel. Os kits WiFi são fáceis de instalar.

    Passo 1: Escolher o Sistema

    WiFi (sem fios): fácil instalação, precisa de bom sinal. PoE (cabo rede): mais fiável, melhor qualidade.

    Passo 2: Posicionar as Câmaras

    Entradas principais, garagem, jardim. A 2.5-3m de altura, ângulo para baixo. Evite contraluz.

    Passo 3: Instalar o Gravador

    NVR (gravador) num local seguro e ventilado. Ligue ao router por cabo ethernet. Instale disco rígido.

    Passo 4: Montar as Câmaras

    Fixe os suportes, passe cabos (PoE) ou configure WiFi. Ajuste o ângulo de visão.

    Passo 5: Configurar

    Instale a app. Configure gravação por movimento, alertas no telemóvel, horários de gravação.

    🛒 Kit 4 câmaras WiFi 2K + NVR 1TB — Amazon.es (~200€)
